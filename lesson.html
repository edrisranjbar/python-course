متغیرها در پایتون

در این جلسه یاد می‌گیری متغیر چیه، چطور تعریف می‌شه و پایتون چطور انواع داده‌های مختلف رو به‌صورت خودکار تشخیص می‌ده.

متغیر چیست؟

متغیر مثل یه جعبه‌ست که می‌تونی یه مقدار توش بذاری و بعداً ازش استفاده کنی. در پایتون نیازی به تعریف نوع متغیر نداری — پایتون خودش می‌فهمه.

Python 
# تعریف متغیر
name = "ادریس"
age = 28
is_student = True

print(name)   # ادریس
print(age)    # 28

قوانین نام‌گذاری

  • می‌تونه شامل حرف، عدد و زیرخط (_) باشه
  • نباید با عدد شروع بشه
  • حساس به بزرگی و کوچکی حروف است — Name و name دو متغیر جداست
  • از کلمات کلیدی پایتون نباید استفاده کرد (مثل if، for، class)
💡
نکته: در پایتون معمولاً از سبک snake_case برای نام‌گذاری استفاده می‌شه — یعنی کلمات با زیرخط از هم جدا می‌شن مثل first_name یا user_age.

تابع type()

برای فهمیدن نوع یه متغیر می‌تونی از تابع type() استفاده کنی:

Python 
x = 10
y = 3.14
z = "سلام"

print(type(x))  # <class 'int'>
print(type(y))  # <class 'float'>
print(type(z))  # <class 'str'>
⚠️
توجه: اگه سعی کنی یه رشته و یه عدد رو با + جمع کنی، پایتون خطا می‌ده. باید اول با str() یا int() تبدیل نوع انجام بدی.

آسان

سه متغیر به اسم‌های first_name، last_name و age تعریف کن و اطلاعات خودت رو توشون بذار. بعد همه‌شون رو با print() چاپ کن.

راهنمایی 💡 
first_name = "نام تو"
last_name = "فامیل تو"
age = ۲۵
print(first_name, last_name, age)
تمرین ۲
متوسط

دو متغیر عددی تعریف کن و عملیات جمع، تفریق، ضرب و تقسیم رو روشون انجام بده و نتیجه هر کدوم رو چاپ کن.

راهنمایی 💡 
a = 10
b = 3
print(a + b)   # جمع
print(a - b)   # تفریق
print(a * b)   # ضرب
print(a / b)   # تقسیم
تمرین ۳
چالشی

یه برنامه بنویس که سن کاربر رو بگیره و سال تولدش رو محاسبه و چاپ کنه. از متغیر و input() استفاده کن.

راهنمایی 💡 
age = int(input("سن تو چنده؟ "))
birth_year = 1403 - age
print("سال تولد:", birth_year)
