قسمت ۳: نوشتن و اجرای اولین برنامه پایتون

پس از نصب و راه‌اندازی پیش‌نیازها، حالا زمان آن رسیده که آستین‌ها را بالا بزنیم و اولین کدهای پایتون را بنویسیم. در این جلسه یاد می‌گیریم چطور در VS Code یک پروژه بسازیم و اولین برنامه‌مان را اجرا کنیم.

ایجاد محیط کاری و ساخت اولین فایل پایتون

برای شروع، ابتدا باید محیط کاری را در VS Code آماده کنیم:

  1. باز کردن VS Code: برنامه‌ای که در جلسه قبل نصب کردیم را باز کنید.
  2. ایجاد و باز کردن پوشه پروژه: روی دکمه Open Folder کلیک کنید یا از منوی File گزینه Open Folder را انتخاب کنید. یک پوشه جدید با نامی انگلیسی مثل codes یا python-course بسازید و آن را باز کنید. انتخاب نام انگلیسی برای پوشه‌ها از بروز خطاهای احتمالی در آینده جلوگیری می‌کند.
  3. ساخت فایل پایتون: در پنل سمت چپ روی فضای خالی کلیک راست کرده و گزینه New File را انتخاب کنید. نام فایل را وارد کنید، مثلاً 1.py یا hello.py. پسوند فایل حتماً باید .py باشد تا سیستم آن را به عنوان کد پایتون بشناسد.

آشنایی با محیط VS Code

محیط این نرم‌افزار از سه بخش اصلی تشکیل شده که هر روز باهاشون سروکار داری:

  • پنل سمت چپ: لیست فایل‌ها و پوشه‌های پروژه را نمایش می‌دهد.
  • پنل وسط (ویرایشگر کد): فضای اصلی کدنویسی است.
  • پنل پایین (ترمینال): خروجی و نتایج اجرای کدها اینجا نمایش داده می‌شود. اگر نمی‌بینیش از منوی View گزینه Terminal را انتخاب کن.

نوشتن برنامه «سلام دنیا» (Hello World)

در دنیای برنامه‌نویسی رسم بر این است که اولین برنامه چاپ عبارت «سلام دنیا» باشد. این کار نمادی از ورود به یک دنیای جدید است.

در ویرایشگر کد این دستور را بنویس:

Python 
print("Hello World")

# یا به فارسی:
print("سلام دنیا")
💡
دستور print چیست؟ کلمه print یعنی نمایش یک مقدار روی صفحه — هیچ ارتباطی با دستگاه پرینتر فیزیکی ندارد! هر متنی که بخواهی به کاربر نشان بدهی را باید داخل پرانتز و علامت نقل‌قول قرار بدهی.

روش‌های اجرای کدهای پایتون

پس از نوشتن کد، دو روش برای اجرا وجود دارد:

  • روش اول — دکمه Run: در گوشه بالا سمت راست VS Code یک دکمه شبیه به دکمه پخش (Play) وجود دارد. با کلیک روی آن کد اجرا شده و خروجی در ترمینال پایین نمایش داده می‌شود.
  • روش دوم — ترمینال: در پنل ترمینال دستور python 1.py را تایپ کن و Enter بزن. به جای 1.py نام فایل خودت را بنویس. این روش در محیط‌های حرفه‌ای کاربرد زیادی دارد.
⚠️
اگر هنگام اجرا با خطای python: command not found مواجه شدی، از دستور python3 1.py استفاده کن — مخصوصاً روی مک و لینوکس.

نتیجه‌گیری

به شما تبریک می‌گویم! اولین کدهای پایتون را نوشتی و اجرا کردی. در این جلسه با محیط VS Code، ساخت فایل‌های .py، مفهوم دستور print و نحوه اجرای کدها آشنا شدیم. در جلسات آینده با متغیرها و انواع داده‌ها کار خواهیم کرد — حتماً این مراحل را روی سیستم خودت تمرین کن.
